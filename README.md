# jamstack-auth-quickstart-frontend
Frontend for JAMstack auth quickstart

## Kinde Custom Domain

* Go to the Admin panel > Settings 
* Click "Custom Domains"
* Click "Add"
* Add a subdomain of your website
    * e.g. "auth.[your-domain]"
* Kinde will give you some CNAME records to add to your domain 
* Add the records and wait
   * Wait started 0738
   * It found my records and status switched from "Verifying" to "Provisioning in progress"
